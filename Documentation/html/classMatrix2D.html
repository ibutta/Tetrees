Detailed Description

template<class Type>
class Matrix2D< Type >

A matrix/vector abstraction class.

Provides an abstraction for handling matrices (multidimensional vectors) and is by far the most important auxiliary piece of code implemented in this project.

Rotate the matrix 90 degrees clockwise.

Applies the transpose operation followed by the reverseRows operation resulting in a 90 degrees clockwise rotation of the matrix.

Remarks
If applied to an array it will result in the reversal of its elements since the transpose of an array has no effect on it.

Reverse matrix's rows or the entire array.

Inverts the order in which the elements of each row of the matrix (or the entire array) appear, i.e. the first element of a row will become its nth, the second its (nth - 1), and so on...
